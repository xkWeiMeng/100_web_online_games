# Day 039 - Pipe Mania

A pipe connection puzzle game where you must connect pipes from the source to the drain before the water flows!

## How to Play

- **Click or tap** grid cells to place pipes
- Pipes are drawn from the queue on the left
- Connect the **SRC** (source) to the **END** (drain) before the countdown ends
- The water starts flowing after a few seconds — build fast!
- Score points for longer pipe paths

## Features

- Progressive difficulty with multiple levels
- Water flow animation with fill effect
- Pipe placement preview on hover
- Mobile touch support
- Score system with level bonuses

## Tech

Pure HTML5 / Canvas — no dependencies. Open `index.html` directly in any browser.
