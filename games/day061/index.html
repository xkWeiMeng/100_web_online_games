How to Play

  • Fill the grid so each row and column contains 1 to N exactly once
  • Bold-outlined groups are cages with a target & operation
  • Numbers in a cage must produce the target using the operation
  • For - and ÷, order doesn't matter—just get the result
  • Single-number cages tell you exactly what goes there
