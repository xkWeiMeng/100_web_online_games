Kakuro

Number Crossword Puzzle

How to Play

  1. Fill blank cells with digits 1 through 9
  2. Clue cells show the required sum for a run of cells
  3. Top-right number = sum downward; Bottom-left = sum across right
  4. No digit may repeat within a single horizontal or vertical run
  5. Use Notes to pencil in candidates (press N on keyboard)
  6. Keyboard: Arrow keys to navigate, 1-9 to enter, Delete to erase
